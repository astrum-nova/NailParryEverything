# Nail Parry Everything
This mod aims to let you nail parry every type of damage in silksong.

# Contributions needed!
While all bosses work, and some enemies work by default by having parryable stuff already,
Many don't, take the grand reed for example, i had to add custom tweaks to the slam and silkstorm.
It is safe to assume there would be much more work on this, but ive been working on this mod for like a week now,
and not only do i want to move on to more projects, but documenting every enemy with their own custom parryable states is not really feasible.

# How to contribute
> Basics
- Enable debug info from the settings
- Enable cheats from the settings
- Read the explanation of what each list does in tweaks.cs
> Level 1
- Make lists or take screenshots of the debug info of something you think should/shouldn't be parryable and send that to me on discord "astrum.nova." with the dot at the end, you can also find me in the hk modding server
> Level 2
- Read level 1 but format the lists according to the formatting in tweaks.cs and add your comment at the top so i can just copy and paste it
> Level 3 (please do this if you can)
- All of the above but open a pr and spam me on discord about it or something idk
